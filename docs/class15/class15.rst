Event-Driven Service Discovery w/ HashiCorp Consul Terraform Sync and BIG-IP
========================================================================================

This is a proctor led intermediate level lab.

In this lab, we will learn how to use F5 BIG-IP AS3 Automation tool along with
Consul Terraform to do Service Discovery and dynamically configure application
objects on BIG-IP. We will learn 

- How to deploy BIG-IP and infrastructure using Terraform
- How to use F5 AS3 Fast template for Service Discovery
- Use Consul Terraform Sync to dynamically update Pool Members
- How to scale / descale applications and update pool Members
- Test End to End HTTP Traffic
